Case 4 — แก้ Widget ใน Preview ("change metric to temperature")

sequenceDiagram actor U as User participant FE as Frontend participant BE as Backend (Go) participant Groq as Groq API participant DB as Database note over U,DB: มี preview dashboard อยู่บนหน้าจอแล้ว U->>FE: "change metric to temperature" FE->>BE: POST /chat { message, conversationId, context: previewData } BE->>DB: INSERT ai_messages (role=user) BE->>DB: SELECT ai_messages (last 8, user+assistant only) note over BE: context != "" → append system msg
"Authoritative current dashboard state…"
buildGroqTools(role) → 14 tools BE->>Groq: POST /completions { messages + context, tools×14 } note over Groq: มี preview context → เห็น widget ปัจจุบัน

preview_update_widget ไม่ query DB เลย — validate แล้ว return changes กลับมา Frontend apply ใน local state เอง
