⚙️ What Drove the Savings

1 Zero-tool gate for greetings ~870 tokens
If the message has no context and no action keywords (metric names, create, add, show…), the full tool list is not sent to Groq. Greetings cost only system prompt + message — no tool schema overhead.
2 Slim tool schemas ~270 tokens
8 of 14 tools use description-only schema (additionalProperties: true). Full JSON property definitions and enums are only sent for 4 complex tools (add/preview_* widgets) where the schema is load-bearing for the model.
3 Preview tools hidden without context ~250 tokens
preview_add_widget, preview_remove_widget, preview_update_widget are excluded from the Groq payload when no dashboard is open. These tools are meaningless without a preview context anyway.
4 Prompt split + context leak fix ~100–845 tokens
System prompt split into base (cache-stable) + context extension (only when dashboard open). A Vue onMounted bug sent stale dashboard context on every request — fixing it eliminated ~845 phantom tokens from "hello" messages after navigation.
