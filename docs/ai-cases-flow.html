💬 "change metric from output to temp"

But graph shows 3 widgets use "output" → ambiguous
Without body.Context AI cannot know which widget the user sees.
SelectTools — builder tools included (graph, not keyword)
OLD: wantsBuilderTools("change metric") → true because "change" is in keyword list.
NEW: graph-driven — len(machines)>0 → builder tools included.
Same result, better reason.
